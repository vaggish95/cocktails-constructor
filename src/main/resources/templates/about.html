КОНСТРУКТОР ЛИЧНЫЙ КАБИНЕТ
О ПРОЕКТЕ

Откуда взялась идея создания сайта? На протяжении четырех лет автор сайта проработал в баре, где ему ежедневно проходилось сталкиваться с необходимостью осуществления рекомендаций гостям ресторана. Однако очень сложно сразу вспомнить все возможные коктейли с теми параметрами, о котором просит посетитель. Именно для решения этой задачи был создан конструктор коктейлей.

Это сайт создан на языке программирования Java и при помощи Spring Boot. Внешнее оформление страницы осуществленно через HTML , CCS и шаблонизатор "Thymeleaf". Среда разработки - IntelliJ IDEA. База данных - Postgres. Связь с базой данных обеспечитвается через фреймворк "Hibernate". Программа запущена на локальном сервера автора, доступ к которому получается через специальный "тоннель" при помощи сайта Ngrok.com. GitHub репозиторий проекта.

В связи с тем, что этот проект создавался исключительно в целях демонстрации навыков программирования, некоторые элементы сайта не проработаны до конца, что может повлиять на качество работы сайта или отображения информации. В настоящий момент продолжается дальнейшая разработка сайта: разрабатывается личный кабинет, возможность пользователям добавлять в базу свои собственные коктейли, улучшаются алгоритмы подбора коктейлей и другое.
